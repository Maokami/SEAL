Detailed Description

Provides the base class for a factory instance that creates instances of UniformRandomGenerator. This class is meant for users to sub-class to implement their own random number generators.

◆ UniformRandomGeneratorFactory() [2/2]

seal::UniformRandomGeneratorFactory::UniformRandomGeneratorFactory ( prng_seed_type  default_seed)
inline

Creates a new UniformRandomGeneratorFactory and sets the default seed to the given value. For debugging purposes it may sometimes be convenient to have the same randomness be used deterministically and repeatedly. Such randomness sampling is naturally insecure and must be strictly restricted to debugging situations. Thus, most users should never have a reason to use this constructor.
