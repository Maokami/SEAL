\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class seal\+::\+Dyn\+Array$<$ T $>$}
A dynamic array for storing objects allocated from a Microsoft SEAL memory pool. The \doxylink{classseal_1_1_dyn_array}{Dyn\+Array} class is mainly intended for internal use and provides the underlying data structure for \doxylink{classseal_1_1_plaintext}{Plaintext} and \doxylink{classseal_1_1_ciphertext}{Ciphertext} classes.

\begin{DoxyParagraph}{Size and Capacity}
\doxylink{classseal_1_1_dyn_array}{Dyn\+Array} allows the user to pre-\/allocate memory (capacity) for the array in cases where the array is known to be resized in the future and memory moves are to be avoided at the time of resizing. The size of the \doxylink{classseal_1_1_dyn_array}{Dyn\+Array} can never exceed its capacity. The capacity and size can be changed using the reserve and resize functions, respectively.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread Safety}
In general, reading from \doxylink{classseal_1_1_dyn_array}{Dyn\+Array} is thread-\/safe as long as no other thread is concurrently mutating it. 
\end{DoxyParagraph}
