\doxysubsection{Detailed Description}
Class to store a public key.

\begin{DoxyParagraph}{Thread Safety}
In general, reading from \doxylink{classseal_1_1_public_key}{Public\+Key} is thread-\/safe as long as no other thread is concurrently mutating it. This is due to the underlying data structure storing the public key not being thread-\/safe.
\end{DoxyParagraph}
