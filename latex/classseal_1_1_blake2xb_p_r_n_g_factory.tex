Creates a new \doxylink{classseal_1_1_blake2xb_p_r_n_g_factory}{Blake2xb\+PRNGFactory} and sets the default seed to the given value. For debugging purposes it may sometimes be convenient to have the same randomness be used deterministically and repeatedly. Such randomness sampling is naturally insecure and must be strictly restricted to debugging situations. Thus, most users should never use this constructor.
