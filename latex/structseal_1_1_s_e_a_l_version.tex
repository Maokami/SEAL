\doxysubsection{Detailed Description}
Holds Microsoft SEAL version information. A \doxylink{structseal_1_1_s_e_a_l_version}{SEALVersion} contains four values\+: \begin{DoxyVerb}1. The major version number;
2. The minor version number;
3. The patch version number;
4. The tweak version number.
\end{DoxyVerb}
 Two versions of the library with the same major and minor versions are fully compatible with each other. They are guaranteed to have the same public API. Changes in the patch version number indicate totally internal changes, such as bug fixes that require no changes to the public API. The tweak version number is currently not used, and is expected to be 0.
