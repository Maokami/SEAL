\doxysubsection{Detailed Description}
Stores a set of attributes (qualifiers) of a set of encryption parameters. These parameters are mainly used internally in various parts of the library, e.\+g., to determine which algorithmic optimizations the current support. The qualifiers are automatically created by the \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext} class, silently passed on to classes such as \doxylink{classseal_1_1_encryptor}{Encryptor}, \doxylink{classseal_1_1_evaluator}{Evaluator}, and \doxylink{classseal_1_1_decryptor}{Decryptor}, and the only way to change them is by changing the encryption parameters themselves. In other words, a user will never have to create their own instance of this class, and in most cases never have to worry about it at all.

Tells whether batching is supported by the encryption parameters. If the plaintext modulus is congruent to 1 modulo 2N, where X\texorpdfstring{$^\wedge$}{\string^}\+N+1 is the polynomial modulus and N is a power of two, then it is possible to use the \doxylink{classseal_1_1_batch_encoder}{Batch\+Encoder} class to view plaintext elements as 2-\/by-\/(N/2) matrices of integers modulo the plaintext modulus. This is called batching, and allows the user to operate on the matrix elements (slots) in a SIMD fashion, and rotate the matrix rows and columns. When the computation is easily vectorizable, using batching can yield a huge performance boost. If the encryption parameters support batching, the variable using\+\_\+batching is set to true. \Hypertarget{classseal_1_1_encryption_parameter_qualifiers_adc14a58c6b2c909b5c808a606fe819b3}\label{classseal_1_1_encryption_parameter_qualifiers_adc14a58c6b2c909b5c808a606fe819b3}
