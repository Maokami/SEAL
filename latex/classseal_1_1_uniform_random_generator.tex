\doxysubsection{Detailed Description}
Provides the base class for a seeded uniform random number generator. Instances of this class are meant to be created by an instance of the factory class \doxylink{classseal_1_1_uniform_random_generator_factory}{Uniform\+Random\+Generator\+Factory}. This class is meant for users to sub-\/class to implement their own random number generators.
