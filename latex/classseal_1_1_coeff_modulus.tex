\doxysubsection{Detailed Description}
This class contains static methods for creating a coefficient modulus easily. Note that while these functions take a sec\+\_\+level\+\_\+type argument, all security guarantees are lost if the output is used with encryption parameters with a mismatching value for the poly\+\_\+modulus\+\_\+degree.

The default value sec\+\_\+level\+\_\+type\+::tc128 provides a very high level of security and is the default security level enforced by Microsoft SEAL when constructing a \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext} object. Normal users should not have to specify the security level explicitly anywhere.
