\doxysubsection{Detailed Description}
Performs sanity checks (validation) and pre-\/computations for a given set of encryption parameters. While the \doxylink{classseal_1_1_encryption_parameters}{Encryption\+Parameters} class is intended to be a light-\/weight class to store the encryption parameters, the \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext} class is a heavy-\/weight class that is constructed from a given set of encryption parameters. It validates the parameters for correctness, evaluates their properties, and performs and stores the results of several costly pre-\/computations.

After the user has set at least the poly\+\_\+modulus, coeff\+\_\+modulus, and plain\+\_\+modulus parameters in a given \doxylink{classseal_1_1_encryption_parameters}{Encryption\+Parameters} instance, the parameters can be validated for correctness and functionality by constructing an instance of \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext}. The constructor of \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext} does all of its work automatically, and concludes by constructing and storing an instance of the \doxylink{classseal_1_1_encryption_parameter_qualifiers}{Encryption\+Parameter\+Qualifiers} class, with its flags set according to the properties of the given parameters. If the created instance of \doxylink{classseal_1_1_encryption_parameter_qualifiers}{Encryption\+Parameter\+Qualifiers} has the parameters\+\_\+set flag set to true, the given parameter set has been deemed valid and is ready to be used. If the parameters were for some reason not appropriately set, the parameters\+\_\+set flag will be false, and a new \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext} will have to be created after the parameters are corrected.

By default, \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext} creates a chain of \doxylink{classseal_1_1_s_e_a_l_context_1_1_context_data}{SEALContext\+::\+Context\+Data} instances. The first one in the chain corresponds to special encryption parameters that are reserved to be used by the various key classes (\doxylink{classseal_1_1_secret_key}{Secret\+Key}, \doxylink{classseal_1_1_public_key}{Public\+Key}, etc.). These are the exact same encryption parameters that are created by the user and passed to th constructor of \doxylink{classseal_1_1_s_e_a_l_context}{SEALContext}. The functions \doxylink{classseal_1_1_s_e_a_l_context_a0ad47117c6c215c73e29ec74c1b43814}{key\+\_\+context\+\_\+data()} and \doxylink{classseal_1_1_s_e_a_l_context_a7cc16cad24e212b1a46cb9216e0356b8}{key\+\_\+parms\+\_\+id()} return the \doxylink{classseal_1_1_s_e_a_l_context_1_1_context_data}{Context\+Data} and the parms\+\_\+id corresponding to these special parameters. The rest of the \doxylink{classseal_1_1_s_e_a_l_context_1_1_context_data}{Context\+Data} instances in the chain correspond to encryption parameters that are derived from the first encryption parameters by always removing the last one of the moduli in the coeff\+\_\+modulus, until the resulting parameters are no longer valid, e.\+g., there are no more primes left. These derived encryption parameters are used by ciphertexts and plaintexts and their respective \doxylink{classseal_1_1_s_e_a_l_context_1_1_context_data}{Context\+Data} can be accessed through the get\+\_\+context\+\_\+data(parms\+\_\+id\+\_\+type) function. The functions \doxylink{classseal_1_1_s_e_a_l_context_a87b32c236c134f844624b40cf9ba32e5}{first\+\_\+context\+\_\+data()} and \doxylink{classseal_1_1_s_e_a_l_context_a50e20dcbc3d1a007aa801785e4c7a33b}{last\+\_\+context\+\_\+data()} return the \doxylink{classseal_1_1_s_e_a_l_context_1_1_context_data}{Context\+Data} corresponding to the first and the last set of parameters in the \"{}data\"{} part of the chain, i.\+e., the second and the last element in the full chain. The chain itself is a doubly linked list, and is referred to as the modulus switching chain.
