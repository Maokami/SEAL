\doxysubsection{Detailed Description}
The \doxylink{classseal_1_1_m_m_prof}{MMProf} is a pure virtual class that every profile for the \doxylink{classseal_1_1_memory_manager}{Memory\+Manager} should inherit from. The only functionality this class implements is the \doxylink{classseal_1_1_m_m_prof_afb1c4d83f66c5575dfdfc80b367f0604}{get\+\_\+pool(mm\+\_\+prof\+\_\+opt\+\_\+t)} function that returns a \doxylink{classseal_1_1_memory_pool_handle}{Memory\+Pool\+Handle} pointing to a pool selected by internal logic optionally using the input parameter of type mm\+\_\+prof\+\_\+opt\+\_\+t. The returned \doxylink{classseal_1_1_memory_pool_handle}{Memory\+Pool\+Handle} must point to a valid memory pool.
