\doxysubsection{Detailed Description}
A memory manager profile that always returns a \doxylink{classseal_1_1_memory_pool_handle}{Memory\+Pool\+Handle} pointing to the new thread-\/safe memory pool. This profile should not be used except in special circumstances, as it does not result in any reuse of allocated memory.
