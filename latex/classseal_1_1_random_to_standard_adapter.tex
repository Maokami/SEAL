\doxysubsection{Detailed Description}
A simple wrapper class to implement C++ Uniform\+Random\+Bit\+Generator type properties for a given polymorphic \doxylink{classseal_1_1_uniform_random_generator}{Uniform\+Random\+Generator} instance. The resulting object can be used as a randomness source in C++ standard random number distribution classes, such as std\+::uniform\+\_\+int\+\_\+distribution, std\+::normal\+\_\+distribution, or any of the standard Random\+Number\+Engine classes.
