\doxysubsection{Detailed Description}
Class to store a secret key.

\begin{DoxyParagraph}{Thread Safety}
In general, reading from \doxylink{classseal_1_1_secret_key}{Secret\+Key} is thread-\/safe as long as no other thread is concurrently mutating it. This is due to the underlying data structure storing the secret key not being thread-\/safe.
\end{DoxyParagraph}
