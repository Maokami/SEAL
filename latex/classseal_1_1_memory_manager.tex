\doxysubsection{Detailed Description}
The \doxylink{classseal_1_1_memory_manager}{Memory\+Manager} class can be used to create instances of \doxylink{classseal_1_1_memory_pool_handle}{Memory\+Pool\+Handle} based on a given \"{}profile\"{}. A profile is implemented by inheriting from the \doxylink{classseal_1_1_m_m_prof}{MMProf} class (pure virtual) and encapsulates internal logic for deciding which memory pool to use.

Returns a \doxylink{classseal_1_1_memory_pool_handle}{Memory\+Pool\+Handle} according to the currently set memory manager profile and prof\+\_\+opt. The following values for prof\+\_\+opt have an effect independent of the current profile\+: \begin{DoxyVerb}mm_prof_opt::force_new: return MemoryPoolHandle::New()

\end{DoxyVerb}
 Other values for prof\+\_\+opt are forwarded to the current profile and, depending on the profile, may or may not have an effect. The value mm\+\_\+prof\+\_\+opt\+::default will always invoke a default behavior for the current profile.
