Detailed Description

Class to store a public key.

Thread Safety
In general, reading from PublicKey is thread-safe as long as no other thread is concurrently mutating it. This is due to the underlying data structure storing the public key not being thread-safe.
See also
KeyGenerator for the class that generates the public key.
SecretKey for the class that stores the secret key.
RelinKeys for the class that stores the relinearization keys.
GaloisKeys for the class that stores the Galois keys.
