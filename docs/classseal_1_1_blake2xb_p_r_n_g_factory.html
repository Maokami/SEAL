seal::Blake2xbPRNGFactory::Blake2xbPRNGFactory ( prng_seed_type  default_seed)
inline

Creates a new Blake2xbPRNGFactory and sets the default seed to the given value. For debugging purposes it may sometimes be convenient to have the same randomness be used deterministically and repeatedly. Such randomness sampling is naturally insecure and must be strictly restricted to debugging situations. Thus, most users should never use this constructor.

Parameters
[in]default_seedThe default value for a seed to be used by all created instances of Blake2xbPRNG
